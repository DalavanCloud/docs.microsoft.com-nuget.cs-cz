<span data-ttu-id="cd9a5-107">Požádejte autora balíčku, aby znovu podepsat pomocí balíčku `nuget sign` příkaz, jak je popsáno v [NuGet docs](https://docs.microsoft.com/en-us/nuget/create-packages/sign-a-package).
