# <a name="nuget-341-release-notes"></a>Zpráva k vydání verze NuGet 3.4.1

[Zpráva k vydání verze NuGet 3.4](../release-notes/nuget-3.4.md) | [zpráva k vydání verze NuGet 3.4.2](../release-notes/nuget-3.4.2.md)

NuGet 3.4.1 byla vydána 30. března 2016 ve stejnou dobu jako Visual Studio 2015 Update 2 a Visual Studio 15 Preview verze řeší několik problémů, které byly zjištěny ve verzi 3.4.

## <a name="updates-and-improvements"></a>Aktualizace a vylepšení

* Opravené problém, který zabránil procházení balíčky z rozhraní Visual Studia při minimální instalaci sady Visual Studio
* Opraven problém s vyhledání sady Visual Studio `lucene.net.dll`
* Všechny zdroje by neměl být výchozí zdroj úložiště po rozšíření NuGet nainstalovat nebo aktualizovat.  Je můžete vyjádřit výslovný souhlas s touto funkcí z nastavení konfigurace.

Pokračujeme v sledování problémů v našem seznamu problémů na Githubu najdete je na: [http://github.com/nuget/home/issues](http://github.com/nuget/home/issues)
